## 🚀 Project

### 🎮 Unity Mirror 기반 전용 서버(Dedicated Server) 구축
> **실시간 네트워크 동기화와 분산 처리 구조 설계**
- **주요 기능:** 매치메이킹, 로비-배틀 서버 분리, 가챠 및 인벤토리 시스템 연동
- **핵심 고민:** Redis를 활용한 세션 관리로 서버 부하 분산 및 실시간 데이터 정합성 확보
- **상세 내용:** [**📄 Notion **](https://www.notion.so/Unity-Dedicated-Server-C-Mirror-2c4dc21508298045b0c0fcc4b95c99aa)
- **소스 코드:** [**📂 GitHub Repository**](https://github.com/ooweaJ/Unity_Dedicate)

## 📌 About Me

- 멀티플레이 게임의 **안정적인 서버 구조 설계**에 깊은 관심이 있습니다.
- 기능 구현을 넘어, 대규모 접속에도 견딜 수 있는 **확장성과 유지보수성**을 최우선으로 고려합니다.
- 고성능 서버 구현을 위해 ECS 및 데이터 지향 설계(DOD)를 학습하며 기술적 깊이를 더하고 있습니다.
- 클라이언트와 인프라를 모두 이해하여, 유기적으로 소통하는 개발자를 목표로 합니다.

---

## 🚀 Focus

- **Dedicated Server:** Unity Mirror 기반의 독립형 서버 구조 설계
- **Architecture:** Matchmaking / Lobby / Battle Server 분리 및 분산 처리
- **Data Management:** Redis를 활용한 실시간 세션 및 게임 상태 관리
- **Infrastructure:** Docker & Kubernetes를 이용한 서버 컨테이너화 및 오케스트레이션
- **Virtualization:** Proxmox 환경에서의 프라이빗 클라우드 인프라 구축 및 운용
- **Contents Logic:** 가챠, 인벤토리 등 게임 핵심 시스템의 서버 사이드 로직 설계
